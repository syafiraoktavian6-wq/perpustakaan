# 📚 Digital Library System (Perpustakaan)

A modern web-based library management system built with **Laravel**, designed to streamline book lending, inventory tracking, and member management.

## 🛠️ Tech Stack
* **Framework:** Laravel 10/11
* **Database:** MySQL / MariaDB (phpMyAdmin)
* **Frontend:** Tailwind CSS / Indigo & Blue Theme
* **Auth:** Multi-role (Admin, Staff, Peminjam)

## 📋 Installation Guide

1. **Clone & Install**
   ```bash
   git clone [https://github.com/syafiraoktavian6-wq/perpustakaan.git](https://github.com/syafiraoktavian6-wq/perpustakaan.git)
   composer install
   npm install && npm run build
